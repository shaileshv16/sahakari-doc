## Component Diagram
- **User Interaction:** Represents the user's interaction with the Mobile App.
- **API Gateway:** Handles the routing of user requests to various components.
- **Services:** Represents different backend services that perform the core business logic.
- **External APIs and Database:** These include external services like NEFT/RTGS, BBPS, Mobile Recharge, and SMS Gateway, along with the database for persistent storage.
